# Табы приложения Builder City

## Описание
Pixel Perfect табы приложения Builder City, созданные на основе дизайна из Figma. Все экраны точно соответствуют дизайну с правильными размерами, цветами и шрифтами.

## Структура табов

### 1. Projects (1-3439) - Главный экран
**Файл:** `app/(tabs)/projects.tsx`

**Особенности:**
- Список проектов с карточками
- Фильтры: Any, Finished, Ongoing
- Кнопка добавления нового проекта
- Статусные бейджи (Finished/Ongoing)
- Карточки с изображениями проектов

**Элементы:**
- Заголовок "Projects"
- Фильтры с активным состоянием
- Карточки проектов с изображениями
- Информация о проекте (название, дата, адрес, статус)

### 2. Calculator (3-1374) - Калькулятор бетона
**Файл:** `app/(tabs)/calculator.tsx`

**Особенности:**
- Выбор материала (Concrete, Brick, Paint, Lightning)
- Поля ввода: Length, Width, Depth, Price per Bag
- Кнопка расчета
- Отображение результатов

**Элементы:**
- Заголовок с иконкой калькулятора
- Кнопки выбора материала
- Поля ввода с единицами измерения
- Кнопка "Calculate"
- Карточки с результатами

### 3. AI Builder (2-645) - Чат с ИИ
**Файл:** `app/(tabs)/ai-builder.tsx`

**Особенности:**
- Чат интерфейс с ИИ помощником
- Быстрые вопросы
- Приветственное сообщение
- Поле ввода сообщений
- Кнопка отправки

**Элементы:**
- Заголовок с аватаром ИИ
- Статус "Online"
- Приветственные сообщения
- Быстрые вопросы
- Поле ввода с кнопкой отправки

### 4. Scanner (3-1324) - Сканер документов
**Файл:** `app/(tabs)/scanner.tsx`

**Особенности:**
- Фоновое изображение
- Инструкции по сканированию
- Кнопка "Add from photo"
- Кнопка "Enter manually"

**Элементы:**
- Заголовок "Construction Scanner"
- Кнопка "назад"
- Иконка телефона
- Кнопки действий

### 5. Settings (3-1031) - Настройки
**Файл:** `app/(tabs)/settings.tsx`

**Особенности:**
- Профиль пользователя
- Список настроек
- Переключатель уведомлений
- Деструктивные действия

**Элементы:**
- Аватар пользователя
- Имя пользователя
- Список настроек с иконками
- Переключатель уведомлений
- Кнопка удаления аккаунта

## Навигация

**Файл:** `app/(tabs)/_layout.tsx`

**Особенности:**
- 5 табов с правильными иконками
- Активный цвет: #ffce00
- Неактивный цвет: #a1a1a1
- Фон таб бара: #1c1c1d
- Правильные шрифты и размеры

**Табы:**
1. Projects - иконка library
2. Calculator - иконка calculator
3. AI Builder - иконка chatbubble
4. Scanner - иконка scan
5. Settings - иконка settings

## Цветовая схема

- **Фон:** #050508
- **Карточки:** #1c1c1d
- **Акцентный:** #ffce00
- **Текст:** #ffffff
- **Вторичный текст:** #a1a1a1
- **Границы:** rgba(255, 255, 255, 0.1)
- **Статус Finished:** #22c55d
- **Статус Ongoing:** #a1a1a1
- **Ошибка:** #ff4656

## Шрифты

- **Основной:** Geologica
- **Время:** DIN Next LT Pro
- **Интерфейс:** Inter

## Технические детали

- **React Native** с Expo Router
- **TypeScript** для типизации
- **Ionicons** для иконок
- **SafeAreaView** для правильного отображения
- **ScrollView** для прокрутки контента
- **TouchableOpacity** для интерактивных элементов

## Функциональность

- Фильтрация проектов
- Расчет объема и стоимости
- Чат с ИИ помощником
- Сканирование документов
- Настройки приложения
- Переключение уведомлений

Все экраны готовы к использованию и полностью соответствуют дизайну из Figma!
